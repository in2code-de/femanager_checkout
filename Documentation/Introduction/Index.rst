.. _what:

What does it do?
----------------

Femanager Checkout is an extension for a TYPO3 Frontend-User Checkout. The idea is, that a user has to pay
for his account after the registration, in order to get access for premium content. Technically it is solved, to add
a new user group to the Frontend User, after a sucessful payment.

.. _requirements:

Requirements of femanager_checkout
----------------------------------

- TYPO3 8.7
- Extbase / Fluid
- femanager 4.2 or newer
